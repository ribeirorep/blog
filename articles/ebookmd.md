Ideal para estudantes, programadores, criadores de conteúdo, freelancers e pessoas que valorizam produtividade.



# Chapters

## Chapter 01 — Introdução ao Markdown

Entenda por que Markdown se tornou um padrão moderno para escrita técnica e criação de conteúdo.

Você vai aprender:

- O que é Markdown
- Por que usar texto simples
- Vantagens sobre editores tradicionais
- Onde Markdown é utilizado hoje
- Como criar um fluxo leve e eficiente



## Chapter 02 — Tutorial Markdown Syntax

Aprenda a sintaxe essencial para escrever documentos organizados e profissionais.

Conteúdo prático:

- Títulos e subtítulos
- Negrito e itálico
- Listas ordenadas e não ordenadas
- Links
- Imagens
- Blocos de código
- Citações
- Tabelas
- Separadores
- Estrutura ideal de documentos



## Chapter 03 — Configurar Extensões Markdown no VS Code

Transforme o Visual Studio Code em um editor poderoso para escrita.

Você vai configurar:

- Preview em tempo real
- Syntax highlight
- Auto format
- Organização de arquivos
- Snippets úteis
- Extensões recomendadas
- Ambiente limpo para produtividade



## Chapter 04 — Instalar e Configurar Pandoc

Aprenda a instalar uma das ferramentas mais poderosas para conversão de documentos.

Passo a passo:

- Download e instalação
- Verificação no terminal
- Configuração inicial
- Estrutura recomendada de pastas
- Teste de funcionamento



## Chapter 05 — Pandoc Workflow + Commands PowerShell

Crie um fluxo real de trabalho no Windows usando PowerShell.

Você vai aprender comandos úteis para:

- Converter Markdown para HTML
- Exportar para PDF
- Exportar para EPUB
- Gerar documentos rapidamente
- Automatizar tarefas repetitivas
- Organizar saídas e versões

Exemplos diretos e prontos para uso.



## Chapter 06 — Export EPUB e PDF Profissional

Transforme seus textos em produtos digitais.

Aprenda:

- Criar eBooks EPUB
- Gerar PDFs limpos
- Ajustar estrutura
- Melhorar aparência final
- Preparar arquivos para venda ou distribuição



## Chapter 07 — Checklist Final, Recursos e Links
